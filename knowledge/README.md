## Rules

- **Public only** — no confidential, proprietary, licensed-restricted, or personal data.
- **Provenance required** — every document needs a source URL and date in the table above.
- **Small is better** — a handful of focused documents keeps indexing fast, cost low, and the demo legible.
- Respect each source's terms of use; include only what those terms permit redistributing, or reference rather than copy where required.
